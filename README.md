## 🔧 自定义指南

### 改变主要颜色

编辑 `index.html` 中的 Tailwind 配置：

```javascript
colors: {
  primary: '#你的颜色代码',      // 改这里改主色
  secondary: '#你的颜色代码',    // 改这里改次色
  accent: '#你的颜色代码',       // 改这里改强调色
}
```

### 改变分类颜色

在 JavaScript 中找到 `getCategoryHoverColor()` 函数，修改映射：

```javascript
'学习工具': 'text-你的颜色类',
```

### 改变过渡速度

全局搜索并替换：

- `transition-all` - 改成 `transition-all duration-500` 变慢
- `duration-200` - 改成其他值

### 添加新分类

1. 在 add form 的 select 中添加 option
2. 在 `getCategoryHoverColor()` 中添加映射
3. 复制一个分类卡片并修改颜色

## 🐛 常见问题

### Q: 如何改变按钮颜色？

A: 找到相应按钮的 class，改 `bg-primary` 为 `bg-你的颜色`

### Q: 深色模式不工作？

A: 检查 body 是否有 `dark-mode` 类

### Q: 删除按钮不显示？

A: 内置链接被禁用，试试添加新网址再删除

### Q: 如何添加更多分类？

A: 在 select 和 `getCategoryHoverColor()` 中添加

### Q: 可以改变字体吗？

A: 在 Tailwind 配置中改 `fontFamily.sans`
